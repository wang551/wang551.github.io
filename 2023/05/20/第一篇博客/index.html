第一篇博客

从很长时间开始，我就想做一个自己的网站，但到最后都半途而废，没能做出来。

终于，今天，它，让我给搞出来了。

以前的时候，总觉得做一个网站很麻烦，需要租服务器，还要装各种软件，所以一直没下手。直到有一次，我发现了静态网页，知道了GitHub Pages、Hexo，只需要把生成的静态页面托管到GitHub上，不用担心服务器的维护和费用，我当时就心动了😎，想着终于可以有一个自己的网站了，可以写博客装逼了。

但是，我还是高估自己了，一个Hexo，就让我晕头转向，在这过程中，我遇到了很多困难，所以一拖再拖，试了几次又放弃了，又过了一段时间心又痒痒的时候，再搞，就这样一次次地重复，每一次都半途而废。

终于，在一个寂静的晚上，我在键盘上敲下hexo deploy，回车，屏幕上输出了：

INFO  Deploy done: git


终于成功部署到GitHub了，接下来就很顺利了，我成功拥有了人生中第一个网站。
